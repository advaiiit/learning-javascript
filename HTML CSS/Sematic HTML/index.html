HTML Layout Elements

  • Main tag is for content unique to this page. Main tag must be used only once per page, and must be put directly inside body tag.

  • Article tag encloses a block of related content that makes sense on its onw without the rest of the page.

  • Section tag is similar to Article tag but is more for grouping together a single part of the page that constitutesone singel piece of functionality or a theme.

  • Aside tag contains content that is not directly related to the main content but can provide additonal in formation indirectly related to it.

  • Header tag represents a group of introductory content. If it is a child of body tag it defines the global header of a webpage, but if it's a child of an article tag or section tag it defines header for that section.

  • nav tag contains the main navigation functionality for the page.

  • Footer represents a group of en content for a page.
